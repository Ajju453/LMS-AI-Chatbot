Setup Guides

Windows Setup Guide
  1. Configure Git:
    • Fill in your username and email in the Git Configuration tab
    • Download or copy the .gitconfig content
    • Save it as .gitconfig in your user home directory
  2. Generate SSH Key:
    • Go to SSH Keys tab and enter your email
    • Click "Generate SSH Key"
    • Copy the public key and add it to code.siemens.com
  3. Test Connection:
    • Open Git Bash or PowerShell
    • Run: ssh -T git@code.siemens.com
    • You should see a success message
  4. Reboot System:
    • Reboot your computer for all components to take effect
Linux/Mac Setup Guide
  1. Configure Git using Terminal:
    • Run: git config --global user.name "Your Name"
    • Run: git config --global user.email "your.email@siemens.com"
  2. Generate SSH Key:
    • Use the SSH Keys tab or run in terminal:
    • ssh-keygen -t ed25519 -C "your.email@siemens.com"
  3. Add SSH Key to Agent:
    • eval "$(ssh-agent -s)"
    • ssh-add ~/.ssh/id_ed25519
  4. Copy Public Key to Git Server:
    • cat ~/.ssh/id_ed25519.pub
    • Copy output and add to code.siemens.com SSH Keys
Troubleshooting
SSH Key Not Working:
  • Ensure the public key is added to code.siemens.com
  • Check file permissions: chmod 700 ~/.ssh
  • Verify key path is correct in .gitconfig
Git Config Not Applied:
  • Restart VSCode and terminal windows
  • Run git config --list to verify settings
  • For Windows, reboot after making changes
Connection Failures:
  • Check internet connection
  • Verify firewall allows SSH (port 22)
  • Test with: ssh -T git@code.siemens.com

Need help? Refer to the Guides tab or contact your administrator.
