# Student Chatbot

A beautiful interactive chatbot application for students to access:
- Personal academic information
- Attendance records
- Course details
- AI-powered Q&A

## How to Run

Just click the "Run" button That's it.

## Test Authentication

Roll Numbers to try: 2223810-2223889
Example: 2223850

## Features

- Student authentication by roll number
- Personal profile and score display
- Course information with TA 1 marks
- 4 months of attendance data
- AI chatbot with Google Gemini
- Beautiful interactive UI
